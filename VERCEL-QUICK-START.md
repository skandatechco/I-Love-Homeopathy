# Vercel Deployment - 3 Commands

## Fastest Way

```bash
# 1. Install Vercel CLI
npm i -g vercel

# 2. Login
vercel login

# 3. Deploy
vercel --prod
```

That's it! Your site is live in 2 minutes.

---

## Or Use GitHub + Web UI

1. Push to GitHub
2. Go to vercel.com
3. Click "Add New Project"
4. Connect your repo
5. Deploy
