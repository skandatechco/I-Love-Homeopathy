PHILOSOPHY: Susceptibility is a name for a state that underlies all possible sickness and all possible cure. Now when a person becomes sick, he becomes susceptible to a certain remedy, which will affect him; while upon a healthy person it will have no effect. Susceptibility exists in the Vital Force, and not in the tissues!

A valuable lesson by Dr. J. T. Kent
